Dynamic Path to R&E Directory

2020-07-28

This blog entry was kindly contributed by the Executive Director of the Office of Research and Evaluation
Please direct all questions to James.Appleton@gcpsk12.org

Purpose

The file, which can be downloaded below serves as a dynamic file path to the ResearchEvaluation root directory on S:\\. It an be updated if the path changes to ensure code written referencing it will not be broken.

File Location

To download and save a file for use, right-click and save this link. The file should be saved in the directory of your choosing (for example C:\\current_path.txt). The ResearchEvaluation path can then be accessed by running

readLines("C:/current_path.txt")
## [1] "S:\\Superintendent\\Private\\Strategy & Performance\\ResearchEvaluation"

or

readLines("C:\\current_path.txt")
## [1] "S:\\Superintendent\\Private\\Strategy & Performance\\ResearchEvaluation"

The path can also be assigned to an object and then used to access the R&E directory. See the following for an example:

re_path <- readLines("C:\\current_path.txt")

paste0(re_path,"\\hi.png")
## [1] "S:\\Superintendent\\Private\\Strategy & Performance\\ResearchEvaluation\\hi.png"
